Test Steps

  1. Look for a small "Feedback" link in the bottom-right corner
  2. Click it to open the recording panel
  3. Click "Start Recording"
  4. Grant browser permissions for screen capture and microphone
  5. Perform some actions or speak (e.g., "This feedback widget is working great!")
  6. Click "Stop & Submit"
  7. Wait for upload to complete
  8. Verify the feedback appears as a Zoho Desk ticket

Embed Code

To add this widget to your own website, use:

<script src="https://feedback-api-production-7e6f.up.railway.app/feedback-widget.js" data-site="your-site-id"></script>

Replace your-site-id with your registered site ID (configured in sites.json).

Configuration

Sites must be registered in the backend's sites.json file. For example:

{ "sites": { "my-website": { "domain": "mywebsite.com", "client_email": "support@mycompany.com", "client_name": "My Company" } } }

Troubleshooting

  • No "Feedback" link appears: The service might not be online yet, or the script tag is not loading correctly.
  • "Chrome only": The widget uses browser APIs (getDisplayMedia) that are only available in Chromium browsers.
  • Permission denied: You must grant screen capture and microphone permissions when the browser prompts.
  • Upload fails: Ensure the backend API is running and accessible.
